# 🐍 Game Ular - fajarskyy

Game klasik Snake yang dibuat menggunakan teknologi web (**HTML, CSS, dan JavaScript**). Tampilannya dibuat modern dengan tema *Dark Mode* dan sentuhan *neon glow*.

## ✨ Fitur Utama
* **Neon Aesthetic:** UI minimalis dengan efek cahaya biru neon yang keren.
* **Responsive Gameplay:** Pergerakan ular yang halus dan responsif.
* **Score Tracker:** Skor otomatis bertambah setiap kali ular makan titik oren.
* **Play & Reset:** Tombol kontrol untuk memulai ulang game dengan cepat.

## 🛠️ Teknologi yang Digunakan
* **HTML:** Untuk struktur dasar dan elemen canvas game.
* **CSS:** Untuk styling tampilan, layout, dan efek visual glow.
* **JavaScript:** Menangani seluruh logika pergerakan, deteksi tabrakan, dan sistem skor.

## 🚀 Cara Menjalankan
Projek ini sangat ringan dan bisa langsung jalan di browser:
1. Clone atau download repositori ini.
2. Buka file `snake.html` menggunakan browser (Chrome, Edge, atau Firefox).
3. Klik tombol **Mulai** dan selamat bermain!

## 🎮 Cara Bermain
Kontrolnya mirip dengan game puzzle/tetris sebelumnya:
* **Tombol Panah (Arrow Keys):** Gunakan Atas, Bawah, Kiri, dan Kanan untuk mengarahkan ular.
* **Tujuan:** Makan titik oren sebanyak-banyaknya untuk memperpanjang ular dan menambah skor tanpa menabrak dinding atau tubuh sendiri.
